# Rescue Wolfy

Top down java based game where you have to find Wolfy before you run out of moves and while watching out for bombs, however there is a twist with the bombs..

# Description

Rescue Wolfy is a java based grid top down game where the player must explore the grid/map and find out where Wolfy is before running out of moves. The player must 
also avoid hidden mines, which are able to move, 50% chance to move when you move on easy and medium, while hard has a 75% chance. Medium and hard mode also has 2 bombs. The game has scuffed animations, and a okay UI.

# How to play?

WASD / arrow controls to move 
F to use the sonar ping/scanner
Avoid bombs on the grid, there are numbers on the top right of a tile that works as a heat signature.
It is able to tell you when bombs get close, and tracks the nearest bomb. If a bomb gets too close,
your heart will start pounding, and you will start breathing heavy. 

Your goal is to find Wolfy without running out of moves or exploding due to the bombs.

# Features

 - Grid based movement system
 - Animated player sprites
 - Scanner that can tell you how far Wolfy and the bomb(s) are
 - Backround music/effects
 - Multiple screens
 - Leaderboard that saves to a file
